# 알고리즘 라이브러리

1~35세대에서 개발·적용된 모든 분석 알고리즘.

## 핵심 알고리즘

### 1. 지수 감쇠 모델 (Exponential Decay)
- **수식:** `P(t) = e^(-λt)`
- **용도:** 시간 경과에 따른 생존 확률 감소
- **입력:** t(경과 시간), λ(감쇠 상수, 재난 유형별)
- **출력:** 생존 확률 (0~1)
- **최초:** 대전 2세대
- **검증:** 1h:50% → 3h:20% → 6h:10% → 12h:3% → 24h:<1%

### 2. 조건부 확률 연쇄 (Event Chain Probability)
- **수식:** `P(최종) = P(A) × P(B|A) × P(C|B)`
- **용도:** 마일스톤 순차 달성 확률
- **입력:** 각 마일스톤의 개별 확률
- **출력:** 최종 이벤트 달성 확률
- **최초:** 대전 3세대
- **예시:** P(수색) = P(소거)×P(진단|소거)×P(수색|진단) = 0.80×0.65×0.60 = 31.2%

### 3. 가우시안 플룸 모델 (Gaussian Plume Dispersion)
- **수식:** `C(x,y,z) = (Q/2πσyσzu) × exp(-y²/2σy²) × [exp(-(z-H)²/2σz²) + exp(-(z+H)²/2σz²)]`
- **용도:** 대기 오염물질 확산 농도
- **입력:** Q(배출량), u(풍속), σ(확산 계수), H(배출 높이)
- **출력:** 거리별 농도 (mg/m³ 또는 μg/m³)
- **최초:** 대전 5세대
- **결과:** 500m:150~200μg/m³, 2km:40~60μg/m³

### 4. ASET/RSET 분석
- **수식:** `안전 판정 = ASET - RSET` (양수=안전, 음수=위험)
- **용도:** 대피 가능 여부 정량 판정
- **입력:** ASET(대피 가능 시간, 연기 도달 기준), RSET(인지+반응+이동 시간)
- **출력:** 안전 마진 (분)
- **최초:** 대전 20세대
- **결과:** ASET(6~9분) - RSET(7~12분) = **-5.2분** (대피 불가능)

### 5. BCR 비용편익 분석 (Benefit-Cost Ratio)
- **수식:** `BCR = 피해방지 편익 / 투자 비용`
- **용도:** 안전 투자 의사결정
- **입력:** 투자 비용, 피해 감소 추정
- **출력:** BCR 배수 (>1이면 투자 가치)
- **최초:** 대전 24세대
- **결과:** 개별 BCR 55.1배, 국가 BCR 2.38배, 통합 BCR 5.27배

### 6. Logistic S-curve (군중 유입 모델)
- **수식:** `N(t) = K / (1 + exp(-r × (t - t_mid)))`
- **용도:** 시간에 따른 군중 유입 곡선
- **입력:** K(포화 인원), r(성장률), t_mid(변곡점)
- **출력:** 시각별 집결 인원
- **최초:** BTS 9세대
- **결과:** K=260,000, r=0.48, t_mid=16:20. **8연속 실측 검증 EXACT MATCH**

### 7. R(t) 위험 곡선 (Risk Trajectory)
- **수식:** `R(t) = Σ(w_i × KPI_i(t)) / Σ(w_i)`
- **용도:** 시간별 종합 위험도 0~10
- **입력:** 7개 KPI 시계열 값 + 가중치
- **출력:** R(t) 값 (GREEN<4, YELLOW 4~7, RED>7)
- **최초:** BTS 17세대
- **결과:** 실측 검증 편차 **0.0% (완벽 일치)**, 3연속 검증

### 8. 화학 반응 경로 (Chemical Reaction Pathway)
- **수식:** 단계별 반응식
- **용도:** 유해물질 반응 메커니즘·생성물 독성
- **최초:** 대전 19세대
- **결과:** Na 3단계 반응 → NaOH 에어로졸 → 이중 질식 메커니즘

### 9. N자형 감성 곡선 (N-Shape Sentiment)
- **수식:** `S(t) = baseline + Σ(phase_amplitude × phase_function(t))`
- **용도:** 대규모 이벤트 참가자 집단 감성 변화
- **입력:** 6개 위상(설렘/집결/피로/감동/긴장/안도)
- **출력:** 감성 값 (-100~+100)
- **최초:** BTS 4세대

### 10. Monte Carlo 정책 시뮬레이션
- **수식:** 확률 분포 기반 반복 시뮬레이션
- **용도:** 정책 시나리오별 미래 사망·비용 추정
- **최초:** 대전 30세대
- **결과:** 현행 유지 5년 사망 30.4명 → 완전 시행 시 9.1명 (-70%)

## 메타 알고리즘

### 11. SMS 시스템 성숙도 (System Maturity Score)
- **수식:** `SMS = Σ(차원_점수 × 가중치)`
- **차원:** 정확도(0.30), 방향(0.20), 커버리지(0.15), 시간예측(0.15), 메타인지(0.20)
- **최초:** 대전 34세대
- **결과:** 84.8점 (B+)

### 12. TRS 이전준비도 (Technology Readiness Score)
- **수식:** 도메인 독립성 × 데이터 범용성 × 구현 용이성
- **최초:** 대전 33세대
- **결과:** 자기 진화 프로토콜 TRS 0.93 (최고)
