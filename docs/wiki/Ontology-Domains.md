# 온톨로지 도메인 정의

> 정의 파일: `data/ontology-domains.json`

## 기본 5개 도메인

### D1 사건 흐름 (Event Flow)

현재 상황이 어떤 단계이며, 다음에 무엇이 예상되는가.

**구성 요소:**
- `phase` — 현재 단계 (주제별 단계 목록 정의)
- `phaseTransition` — 다음 단계 전환 조건·예상 시점
- `causalChain` — 원인→중간사건→결과 인과 체인
- `timeline` — 주요 이벤트 시간순 배치

**단계 예시:**
- 재난: 발화→확산→최성기→진압→잔불→수색/복구
- 이벤트: 준비→세팅→집결→진행→종료→해산

**알고리즘:** 이벤트 체인 확률 분석 `P(최종) = P(A) × P(B|A) × P(C|B)`

---

### D2 공간 영향 (Spatial Impact)

피해/영향 범위, 확산 가능성, 주변 지역 영향.

**구성 요소:**
- `epicenter` — 중심 좌표
- `impactZones` — 직접/간접/잠재 영향권
- `geographicFactors` — 지형, 건물, 인접 시설
- `spreadPath` — 확산 경로 (풍향, 이동 패턴)

**알고리즘:**
- 가우시안 플룸 (대기 확산): `C(x,y,z) = (Q/2πσyσzu) × exp(...)`
- 밀집도 구역 모델: `ρ = N/A [인/m²]`, Keith Still 기준 5인/m² 위험

---

### D3 투입 자원 (Resource Deployment)

현재 대응 체계의 적정성과 추가 필요 자원.

**구성 요소:**
- `deployed` — 투입 인력·장비·시설
- `adequacy` — 투입-수요 갭
- `fatigue` — 인력 피로도, 교대 필요성
- `additionalNeeds` — 추가 필요 자원

**알고리즘:** BCR 비용편익 분석 `BCR = 편익/비용` (BCR>1 투자 가치)

---

### D4 인명 피해 (Casualty Analysis)

현재 현황, 추이, 구조/위험 가능성.

**구성 요소:**
- `current` — 사망, 부상, 실종, 관련 인원
- `trend` — 시간 추이
- `goldenTime` — 골든타임, 생존 확률 곡선
- `longTerm` — PTSD, 장해, 직업 복귀

**알고리즘:**
- 지수 감쇠: `P(t) = e^(-λt)` — 시간별 생존 확률
- ASET/RSET: `ASET < RSET` → 대피 불가능
- N자형 감성 곡선: 집단 감성의 시간적 변화

---

### D5 2차 피해 (Secondary Hazards)

파생 위험, 유해물질, 환경·사회 영향.

**구성 요소:**
- `hazardousMaterials` — 유형, 양, 반응 경로
- `structuralRisk` — 붕괴, 추가 사고
- `environmental` — 대기, 토양, 수질
- `social` — 여론, 경제, 법적 영향

**알고리즘:**
- 화학 반응 경로: 반응물→생성물 단계별 체인
- R(t) 위험 곡선: `R(t) = Σ(w_i × KPI_i) / Σ(w_i)` — 7 KPI 가중 평균

---

## 도메인 간 관계

```
D1(사건) ──causes──→ D4(인명) ──demands──→ D3(자원) ──mitigates──→ D1
     │ triggers                    ↑ constrains
     ↓                             │
    D5(2차) ──expands──→ D2(공간) ─┘
     └──feedback──→ D1 (악순환 가능)
```

| 관계 | 의미 | 예시 |
|------|------|------|
| causes | 사건→인명 피해 | 화재→사망 11명 |
| triggers | 사건→2차 피해 | 화재→나트륨 반응 |
| demands | 인명→자원 요구 | 부상 59명→의료팀 |
| mitigates | 자원→사건 완화 | 소방 115명→진압 |
| expands | 2차→공간 확대 | 유해가스→3km 확산 |
| constrains | 공간→인명 제약 | 4중 함정→대피 불가 |
| feedback | 2차→사건 악화 | 나트륨 폭발→재발화 |

## 확장 도메인 (E1~E6)

기본 5개 외에 주제·세대에 따라 선택 적용.

| ID | 이름 | 설명 | 최초 사용 |
|----|------|------|----------|
| E1 | 법적 분석 | 중대재해법, 산재, 민사 | 대전 18세대 |
| E2 | 국제 비교 | 글로벌 유사 사례 | 대전 21세대 |
| E3 | 정책 설계 | 로드맵, 시뮬레이션 | 대전 27세대 |
| E4 | 네트워크 | 다계층 영향 분석 | BTS 3세대 |
| E5 | 실시간 | KPI, CTP, 라이브 검증 | BTS 11세대 |
| E6 | 메타 분석 | 시스템 성능, 편향 | 대전 7세대 |
