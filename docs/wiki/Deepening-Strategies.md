## 새 전략 창안 가이드

기본 12개가 소진되면:
1. 기존 전략들을 **조합**한다 (D1+D2 → 복합 실패 메커니즘)
2. 기존 전략을 **반전**한다 (분석→역설계, 과거→미래 시뮬)
3. 상위 차원으로 **통합**한다 (개별→종합, 분석→교훈)
4. **인접 분야**를 도입한다 (법률, 심리, 경제, 국제)
5. **시스템 자체**를 분석한다 (메타인지, 적중률, 진화 감사)
